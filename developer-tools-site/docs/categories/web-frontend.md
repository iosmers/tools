## 22. 前端、浏览器与 Web 调试

| 工具 | 用途 | 适合场景 | 获取/安装 |
| --- | --- | --- | --- |
| Google Chrome | DevTools 生态 | 主流 Web 调试 | `brew install --cask google-chrome` |
| Firefox Developer Edition | Firefox 开发版 | CSS/Grid/兼容性测试 | `brew install --cask firefox@developer-edition` |
| Safari Technology Preview | Safari 新特性 | iOS/macOS Safari 兼容性 | Apple 官网 |
| Arc | 现代浏览器 | 多空间、标签组织 | 官网 |
| Responsively App | 多尺寸响应式预览 | 前端布局测试 | `brew install --cask responsively` |
| Polypane | 多视口/无障碍/SEO | 专业前端测试 | 官网 |
| Sizzy | 多设备预览 | 前端 UI 测试 | 官网 |
| BrowserStack | 云浏览器测试 | 多设备兼容性 | Web |
| Lighthouse | 性能/SEO/可访问性 | Web 性能审计 | Chrome/`npm` |
| WebPageTest | Web 性能测试 | 真实网络性能 | Web |
| Playwright | 浏览器自动化测试 | E2E 测试 | `npm init playwright` |
| Cypress | E2E 测试 | 前端测试 | npm |
| Storybook | 组件文档/测试 | 组件库 | npm |
| Vite | 前端构建 | 现代前端项目 | npm |
