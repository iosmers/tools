## 10. 窗口、桌面、启动器与菜单栏效率 {#window-desktop}

### 启动器 / 命令中心

| 工具 | 核心优势 | 适合人群 | 获取方式 |
| --- | --- | --- | --- |
| Raycast | 插件生态、命令中心、剪贴板、窗口管理、AI | 现代 Mac 效率首选 | `brew install --cask raycast` |
| Alfred | 老牌启动器、工作流成熟 | 重度自动化/老用户 | 官网 |
| LaunchBar | 快速启动和动作 | 传统效率用户 | 官网 |
| Spotlight | 系统自带 | 轻量搜索应用/文件 | 系统自带 |

### 窗口管理

| 工具 | 类型 | 核心优势 | 安装/获取 |
| --- | --- | --- | --- |
| Rectangle | 快捷键分屏 | 免费开源，上手简单 | `brew install --cask rectangle` |
| Magnet | 拖拽/快捷键分屏 | 简单稳定 | App Store |
| Moom | 自定义布局 | 更精细的窗口布局 | 官网 |
| BetterSnapTool | 拖拽吸附 | 老牌窗口吸附 | App Store |
| AeroSpace | 平铺窗口管理 | 类 i3、配置文件、键盘流 | `brew install --cask nikitabobko/tap/aerospace` |
| yabai + skhd | 高度可编程平铺 | 极客/自动化 | `brew install yabai skhd` |
| Amethyst | 自动平铺窗口 | 想要开箱即用平铺 | `brew install --cask amethyst` |
| AltTab | Windows 风格窗口切换 | 多窗口预览切换 | `brew install --cask alt-tab` |
| Hammerspoon | Lua 自动化 | 自定义窗口规则/快捷键 | `brew install --cask hammerspoon` |
