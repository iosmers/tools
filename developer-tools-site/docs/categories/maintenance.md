## 27. 维护建议

- 每个工具至少保留：**功能、核心优势、适合场景、安装方式、官网**。
- 对商业工具增加：**价格/授权/免费限制待查** 标记，避免过期。
- 对安全敏感工具增加：**权限、证书、密钥存储、数据上传** 说明。
- 大列表后续可以拆成多个页面：`terminal.md`、`network.md`、`database.md`、`macos.md`、`security.md`、`ai.md`。
- 后续可用 YAML/JSON 维护工具数据，再自动生成 Markdown 表格和标签页。
