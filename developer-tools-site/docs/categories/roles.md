## 25. 按角色推荐

### 后端工程师

- 终端：Ghostty / WezTerm、tmux/Zellij、Starship、Atuin、zoxide、fzf。
- 代码：VS Code / JetBrains、ripgrep、fd、bat、delta、lazygit。
- API：HTTPie/xh、Bruno/Postman、Proxyman/mitmproxy。
- 数据库：TablePlus / DBeaver / DataGrip、pgcli、mycli、redis-cli。
- 容器：OrbStack / Docker Desktop、lazydocker、dive、k9s。

### 前端工程师

- 浏览器：Chrome、Firefox Developer Edition、Safari Technology Preview。
- 调试：Proxyman、Responsively、Polypane/Sizzy、Lighthouse、Playwright。
- 图片：ImageOptim、Squoosh、ImageMagick、Shottr/CleanShot X。
- 包管理：pnpm、Bun、Deno、Volta/fnm。
- 文档：Storybook、VitePress/Docusaurus。

### AI / 数据工程师

- Python：uv、ruff、mise、direnv、just。
- 本地模型：Ollama、LM Studio、llama.cpp、Open WebUI。
- 远程/GPU：SSH、tmux、mosh、btop、nvtop/服务器监控、rclone、JuiceFS。
- 数据：DuckDB、Miller、jq、yq、DBeaver、TablePlus。
- 实验记录：Obsidian、Jupyter、Marimo、Quarto。

### DevOps / 平台工程

- 云：awscli、gcloud、az、doctl、terraform/OpenTofu、Pulumi、Ansible。
- K8s：kubectl、k9s、kubectx/kubens、Helm、stern、Lens/OpenLens。
- 安全：1Password/Bitwarden、sops、age、Vault、gitleaks、trivy、syft、cosign。
- 网络：mtr、iperf3、nmap、Wireshark、Tailscale、cloudflared、frp。
- 监控：btop、glances、Stats、iStat Menus、Little Snitch/LuLu。

### 远程服务器重度用户

- 连接：OpenSSH、mosh、autossh、Termius/Royal TSX/Tabby。
- 会话：tmux、Zellij。
- 文件：Yazi、nnn、rclone、rsync、restic。
- 排障：htop/btop、procs、lsof、duf/dust/ncdu、lnav、doggo、mtr。
