# 数据库、缓存和数据客户端

SQL/NoSQL 图形客户端、Redis/Mongo 官方工具，以及数据库 CLI。

!!! note "维护方式"
    本页是工具百科的一个专题页。新增工具时，请优先补充到对应功能表格中，并保留：核心优势、适合场景、安装方式、官方链接。


## 15. 数据库、缓存和数据客户端 {#databases}

### GUI 数据库客户端

| 工具 | 覆盖范围 | 核心优势 | 获取方式 |
| --- | --- | --- | --- |
| TablePlus | MySQL/PostgreSQL/SQLite/Redis 等 | 原生、轻量、颜值高 | `brew install --cask tableplus` |
| DBeaver | SQL/NoSQL/大量 JDBC 数据源 | 免费开源版、覆盖广 | `brew install --cask dbeaver-community` |
| DataGrip | JetBrains 数据库 IDE | SQL 智能、专业开发 | JetBrains Toolbox |
| Navicat | 多数据库商业客户端 | 老牌、企业常见 | 官网 |
| Beekeeper Studio | SQL 客户端 | 开源/现代 UI | `brew install --cask beekeeper-studio` |
| Sequel Ace | MySQL/MariaDB | 免费 Mac MySQL 客户端 | App Store |
| Postico | PostgreSQL | Mac 原生 Postgres | 官网 |
| pgAdmin | PostgreSQL 官方管理工具 | 管理功能完整 | `brew install --cask pgadmin4` |
| Redis Insight | Redis 官方 GUI | Redis 可视化和分析 | 官网 |
| MongoDB Compass | MongoDB 官方 GUI | Mongo 查询/索引/聚合 | `brew install --cask mongodb-compass` |
| Studio 3T | MongoDB 商业客户端 | Mongo 高级功能 | 官网 |
| DB Browser for SQLite | SQLite GUI | 小型本地库查看 | `brew install --cask db-browser-for-sqlite` |

### CLI 数据库工具

| 工具 | 用途 | 安装 |
| --- | --- | --- |
| usql | 多数据库 SQL CLI | `brew install xo/xo/usql` |
| pgcli | PostgreSQL 自动补全 CLI | `brew install pgcli` |
| mycli | MySQL 自动补全 CLI | `brew install mycli` |
| litecli | SQLite 自动补全 CLI | `brew install litecli` |
| mongosh | MongoDB Shell | `brew install mongosh` |
| redis-cli | Redis CLI | `brew install redis` |
| duckdb | 嵌入式分析数据库 | `brew install duckdb` |
| clickhouse-client | ClickHouse 客户端 | `brew install clickhouse` |
