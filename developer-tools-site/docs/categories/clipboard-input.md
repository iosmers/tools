## 11. 剪贴板、文本片段与输入增强 {#clipboard}

| 工具 | 类型 | 核心优势 | 适合场景 | 获取方式 |
| --- | --- | --- | --- | --- |
| Maccy | 剪贴板历史 | 开源、轻量、键盘优先 | 代码/命令反复复制 | `brew install --cask maccy` |
| Paste | 剪贴板历史库 | UI 好、长期保存、分类 | 重度剪贴板用户 | Setapp/App Store |
| Raycast Clipboard | 启动器内剪贴板 | 和命令中心整合 | Raycast 用户 | Raycast 内置 |
| Alfred Clipboard | 启动器内剪贴板 | 和 Alfred Workflow 整合 | Alfred 用户 | Alfred Powerpack |
| CopyQ | 跨平台剪贴板 | 开源、脚本化 | 多平台用户 | `brew install --cask copyq` |
| Flycut | 程序员剪贴板 | 简单、轻量 | 代码片段复制 | `brew install --cask flycut` |
| Clipy | 开源剪贴板 | 简单历史和片段 | 免费替代 | GitHub |
| Espanso | 文本扩展 | 跨平台文本片段 | 邮件模板、代码模板 | `brew install espanso` |
| TextExpander | 商业文本扩展 | 团队片段共享 | 客服/团队模板 | 官网 |
| Karabiner-Elements | 键盘映射 | 改键、复杂规则 | 键盘流/程序员 | `brew install --cask karabiner-elements` |
