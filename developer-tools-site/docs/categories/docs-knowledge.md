# 笔记、文档、知识库与图表

本地 Markdown 知识库、云端协作文档、文档站框架、UML 和架构图工具。

!!! note "维护方式"
    本页是工具百科的一个专题页。新增工具时，请优先补充到对应功能表格中，并保留：核心优势、适合场景、安装方式、官方链接。


## 20. 笔记、文档、知识库与图表

| 工具 | 类型 | 核心优势 | 适合场景 | 获取方式 |
| --- | --- | --- | --- | --- |
| Obsidian | 本地 Markdown 知识库 | 双链、插件、本地文件 | 个人知识库、技术笔记 | `brew install --cask obsidian` |
| Logseq | 大纲/双链知识库 | 隐私优先、开源、日志流 | 日记、任务、会议记录 | 官网/GitHub |
| Joplin | 开源笔记 | 端到端同步、Markdown | 开源替代 Evernote | `brew install --cask joplin` |
| Notion | 云端协作文档 | 数据库、团队协作 | 项目管理、团队文档 | `brew install --cask notion` |
| Zettlr | Markdown 写作 | 学术/长文档 | Markdown 作者 | `brew install --cask zettlr` |
| Typora | Markdown 编辑器 | 所见即所得 | 写文档/博客 | 官网 |
| MarkText | 开源 Markdown 编辑 | 免费替代 Typora | Markdown 写作 | GitHub |
| MkDocs | 文档站 | Python/Markdown 文档 | 项目文档、Read the Docs | `pip install mkdocs` |
| Material for MkDocs | MkDocs 主题 | 搜索、导航、深色模式 | 美观文档站 | `pip install mkdocs-material` |
| Docusaurus | React 文档站 | 版本化、插件生态 | 前端/开源项目文档 | npm |
| VitePress | Vue/Vite 文档站 | 快速、简洁 | 前端文档 | npm |
| Mermaid | 文本画图 | 流程图/时序图 | 文档图表 | `brew install mermaid-cli` |
| PlantUML | UML 图 | 架构/序列图 | 企业文档 | `brew install plantuml` |
| draw.io / diagrams.net | 图形画图 | 架构图、流程图 | 通用画图 | `brew install --cask drawio` |
| Excalidraw | 手绘风白板 | 架构草图 | 讨论/演示 | Web |
