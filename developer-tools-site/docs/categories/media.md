## 21. 截图、录屏、图片和媒体处理

| 工具 | 类型 | 核心优势 | 适合场景 | 获取方式 |
| --- | --- | --- | --- | --- |
| Shottr | 截图/OCR/贴图 | 轻量、OCR、滚动截图、标注 | 开发截图、取色、OCR | 官网 |
| CleanShot X | 专业截图录屏 | 录屏、云分享、标注、贴图 | 专业内容制作 | Setapp/官网 |
| Snagit | 截图录屏 | 企业文档、教程 | 教程/培训 | 官网 |
| macOS Screenshot | 系统截图 | 无需安装 | 基础截图 | 系统自带 |
| Kap | GIF/录屏 | 开源录屏转 GIF | 开源演示 | `brew install --cask kap` |
| Screen Studio | 高质量录屏 | 自动缩放、产品视频 | 产品演示 | 官网 |
| OBS Studio | 录屏/直播 | 开源强大 | 直播、课程 | `brew install --cask obs` |
| Gifski | 视频转高质量 GIF | README/GitHub 动图 | `brew install gifski` |
| ImageOptim | 图片无损压缩 | 前端图片优化 | `brew install --cask imageoptim` |
| Squoosh | Web 图片压缩 | 格式转换、对比 | Web |
| ImageMagick | CLI 图片处理 | 批量 resize/convert | `brew install imagemagick` |
| FFmpeg | 音视频处理 | 转码、裁剪、抽帧 | `brew install ffmpeg` |
| HandBrake | 视频转码 GUI | 视频压缩 | `brew install --cask handbrake` |
| Permute | 媒体格式转换 | 商业批量转换 | Setapp/官网 |
| XnConvert | 批量图片转换 | 图片批处理 | 官网 |
