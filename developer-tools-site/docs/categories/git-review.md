## 13. Git、代码评审与版本控制 {#git}

| 工具 | 类型 | 核心优势 | 安装/获取 |
| --- | --- | --- | --- |
| Git | VCS | 标准版本控制 | `brew install git` |
| GitHub CLI `gh` | 平台 CLI | PR/Issue/Actions/Release | `brew install gh` |
| GitLab CLI `glab` | 平台 CLI | MR/Issue/CI | `brew install glab` |
| lazygit | Git TUI | 终端交互式 Git | `brew install lazygit` |
| gitui | Git TUI | Rust、轻量快速 | `brew install gitui` |
| tig | Git 历史 TUI | 查看 commit/diff/log | `brew install tig` |
| delta | Git diff pager | diff 高亮 | `brew install git-delta` |
| difftastic | 结构化 diff | 按语法树比较代码 | `brew install difftastic` |
| pre-commit | Git hooks | 提交前自动检查 | `brew install pre-commit` |
| lefthook | Git hooks | 快速、多语言 hook | `brew install lefthook` |
| git-lfs | 大文件版本管理 | 模型/数据/二进制文件 | `brew install git-lfs` |
| git-crypt | Git 加密文件 | 小规模密钥文件加密 | `brew install git-crypt` |
| Fork | GUI Git | Mac 原生、交互好 | 官网 |
| Tower | GUI Git | 专业商业 Git 客户端 | 官网 |
| GitKraken | GUI Git | 分支图、团队功能 | 官网 |
| Sourcetree | GUI Git | 免费、基础完整 | 官网 |
| GitHub Desktop | GUI Git | GitHub 新手友好 | `brew install --cask github` |
| Kaleidoscope | Diff/Merge | 商业文件对比 | 官网/Setapp |
| Beyond Compare | Diff/Merge | 跨平台强对比 | 官网 |
