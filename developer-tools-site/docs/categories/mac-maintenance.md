## 24. Mac 系统维护、卸载和应用管理

| 工具 | 用途 | 适合场景 | 获取方式 |
| --- | --- | --- | --- |
| Homebrew Bundle | 用 Brewfile 管理应用 | 新机器初始化 | `brew bundle` |
| mas | Mac App Store CLI | 脚本安装 App Store 应用 | `brew install mas` |
| AppCleaner | 卸载应用残留 | 免费清理 App | `brew install --cask appcleaner` |
| Pearcleaner | 开源卸载清理 | AppCleaner 替代 | GitHub/Homebrew |
| Latest | 检查应用更新 | 非 App Store 应用更新提醒 | `brew install --cask latest` |
| MacUpdater | 应用更新管理 | 商业更新检查 | 官网 |
| OnyX | 系统维护 | 缓存/维护脚本 | 官网 |
| CleanMyMac | 商业清理维护 | 一体化清理 | 官网/Setapp |
| Suspicious Package | 查看 pkg 内容 | 安装包安全检查 | `brew install --cask suspicious-package` |
| Pearcleaner/AppCleaner + Brewfile | 组合 | 保持系统可复现 | - |
