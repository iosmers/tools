Homebrew、Nix、多语言版本管理、Node/Python/Rust/Go/Java 等开发环境管理。

## 18. 包管理、语言版本与项目环境

| 方向 | 工具 | 用途 | 安装 |
| --- | --- | --- | --- |
| Mac 包管理 | Homebrew | 安装 CLI/GUI 应用 | 官网脚本 |
| Mac 包管理 | MacPorts | Homebrew 替代 | 官网脚本 |
| 可复现环境 | Nix | 声明式包和开发环境 | `brew install nix` 或官方脚本 |
| 多语言版本 | mise | Python/Node/Go/Rust 等版本和任务 | `brew install mise` |
| 多语言版本 | asdf | 多语言版本管理 | `brew install asdf` |
| Node 版本 | fnm、nvm、Volta | Node 版本管理 | `brew install fnm nvm volta` |
| JS 运行时 | Node.js、Bun、Deno | 前端/脚本运行时 | `brew install node bun deno` |
| JS 包管理 | pnpm、yarn、npm | 前端依赖 | `brew install pnpm yarn` |
| Python 项目 | uv | Python 包/项目/venv/版本 | `brew install uv` |
| Python 包隔离 | pipx | 安装 Python CLI | `brew install pipx` |
| Python 数据环境 | mamba/micromamba | Conda 生态快速替代 | `brew install micromamba` |
| Rust | rustup、cargo | Rust 工具链 | `brew install rustup-init` |
| Go | go、gvm | Go 工具链 | `brew install go` |
| Java | SDKMAN、jenv | JDK 版本管理 | `brew install jenv` |
| Ruby | rbenv、chruby | Ruby 版本 | `brew install rbenv chruby` |
| PHP | phpbrew、composer | PHP 版本/依赖 | `brew install composer` |
