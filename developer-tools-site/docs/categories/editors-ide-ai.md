## 12. 编辑器、IDE 与 AI 编程工具 {#editors-ide-ai}

| 工具 | 类型 | 核心优势 | 适合人群 | 获取方式 |
| --- | --- | --- | --- | --- |
| VS Code | 通用编辑器 | 插件生态、Remote、Dev Container、Copilot | 大多数开发者 | `brew install --cask visual-studio-code` |
| JetBrains Toolbox | IDE 管理器 | PyCharm/GoLand/CLion/IDEA/DataGrip 等 | 专业语言 IDE 用户 | `brew install --cask jetbrains-toolbox` |
| Zed | 现代代码编辑器 | 高性能、协作、AI 方向 | 追求速度和新体验 | `brew install --cask zed` |
| Cursor | AI 代码编辑器 | 代码库理解、多文件 AI 编辑 | AI 辅助开发 | 官网 |
| Windsurf | AI IDE | Agentic IDE 工作流 | AI 编程用户 | 官网 |
| Sublime Text | 高性能文本编辑器 | 启动快、大文件强 | 轻量编辑 | `brew install --cask sublime-text` |
| Nova | Mac 原生编辑器 | Panic 出品，原生 UI | Mac 原生体验爱好者 | 官网 |
| Neovim | 终端编辑器 | LSP/treesitter/插件化 IDE | Vim/SSH 用户 | `brew install neovim` |
| Helix | 终端编辑器 | 内置 LSP/treesitter，默认配置强 | 想要现代 Vim 替代 | `brew install helix` |
| Emacs | 可编程编辑环境 | Org-mode、Lisp、极致扩展 | 深度定制用户 | `brew install emacs` |
| Vim | 经典终端编辑器 | 几乎所有服务器都有 | 远程维护 | 系统自带/`brew install vim` |

### 编辑器插件方向

| 方向 | VS Code / 编辑器常用选择 |
| --- | --- |
| AI 辅助 | GitHub Copilot、Continue、Codeium、Tabnine、Cody |
| 远程开发 | Remote SSH、Dev Containers、WSL（跨平台）、JetBrains Gateway |
| Python | Python、Pylance、Ruff、Jupyter |
| 前端 | ESLint、Prettier、Vue/React/Svelte 扩展、Tailwind CSS IntelliSense |
| Markdown | Markdown All in One、Mermaid、Paste Image |
| Git | GitLens、Git Graph、GitHub Pull Requests |
